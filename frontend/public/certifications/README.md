# Certificate Images

This folder contains certificate images for the portfolio.

## How to add certificate images:

1. Save your certificate images in this folder
2. Use the following naming convention:
   - prompt_engineering.png (for Prompt Engineering certificate)
   - generative_ai.png (for Generative AI certificate)
   - fao_science.png (for FAO Science certificate)
   - gamification.png (for Gamification certificate)
   - system_verilog.png (for System Verilog certificate)
   - tannery_effluents.png (for Tannery Effluents certificate)

3. The images will automatically appear in the "View Certificate" modal on the Experience page.
